  ### HTTP Request

  `GET [site_url]/wp-json/wcmp/v1/vendors/[vendor_id]/reviews`

  ### Query Parameters

  Parameter  | Description
  ---------  | -----------
   vendor_id | Unique identifier for the vendor
   review_id | Unique identifier for a review
